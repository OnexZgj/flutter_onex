# flutter_onex

A new Flutter application.


### 1、应用介绍:
基于[WanAndroid](http://www.wanandroid.com/ "WanAndroid")开放[API](http://www.wanandroid.com/blog/show/2 "API")开发的技术文章阅读App。
该应用程序是玩Android部分功能的flutter版本和干货网站部分api。

- 主要功能包括：首页、项目、公众号、搜索等。
